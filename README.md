# Гвинт
Мультиплеер игра Гвинт (версия из Ведьмак 3) с возможностью регистрации и сохранения собранной колоды. Игра оптимизирована под соотношение сторон 16:9, для лучшего игрового опыта советуем включить полноэкранный режим при помощи клавиши F11. В игре реализованы: 4 фракции с их особенностями, 16 лидеров и их уникальные способоности, 149 карт, 14 способностей карт.

Стек: TypeScript, Vue3, Pinia, Vite, SCSS, WebSocket, REST API, ESlint, Prettier.

Главная страница содержит возможность залогиниться/зарегистрироваться, прочитать правила и посмотреть авторов.

## Регистрация 
  - Можно ввести имя пользователя, должно содержать не менее 4 символов 
  - Пароль, должен быть не менее 6 символов, содержать хотя бы 1 цифру

## Авторизация 
  - Реализуется вход по созданным данным 
  - Авторизованный пользователь при повторном посещении страницы не проходит повторную авторизацию 

## Главная страница 
  - Есть возможность выйти из своего аккаунта 
  - Footer 
    - содержит ссылку на гитхабы
    - содержит ссылку на сайт школы
  - На другие страницы игры (/deck и /game) можно перейти только с главной страницы (только через кнопки в игре, а не написав соответсвующую ссылку в адресной строке)

## Страница сборки колоды 
  - Сохраняются колоды за авторизованным пользователем (одна колода, которая отображается справа на каждую фракцию, колода сохраняется при нажатии кнопки “Начать игру” если эта кнопка проходит валидацию. Если вы перезагрузите страницу или решите поиграть спустя время, ваша колода загрузиться с сервера) 
  - Выбор фракции (4 фракции: Северные Королевства, Империя Нильфгаард, Скоя'таэли, Чудовища) 
  - Выбор лидера фракции 
  - Верное отображение всех карт фракции и нейтральных карт. 
    - у каждой фракции есть свой набор фракционных карт и нейтральных карт 
    - нейтральные карты для каждой фракции уникальны по количеству отображения в левом и правом окне 
    - Каждую карту можно посмотреть с описанием нажав на нее правую кнопку мыши 
  - Отображение общих данных о вашей колоде (Карт в колоде, карт отрядов, специальные карты, общая сила карт отрядов, количество героев в колоде) 
  - Фильтрация карт по отрядам(ближних, стрелков и катапульт), героям, специальным картам и картам погоды 
  - Возможность составления колоды (ограничения по количеству карт в колоде нет) 
    - колода содержит 22 или более карт отряда
    - колода содержит до 10 спец карт
  - Переход на страницу с игрой, при переходе проверяется соответствует ли колода требованиям 
  - Окно поиска игры (один игрок не может подключиться к игре, идет ожидание второго игрока и только тогда они переходят на страницу с игрой)

## Страница с игрой 
  - Отображается счет каждого игрока  (лидирующий по очкам выделяется золотым венком) 
    - отображается общий счет игрока 
    - отображается счет каждого поля игрока 
  - Указана рядом с иконкой игрока информация о нем 
    - показано количество жизней (2 рубина) 
    - отображается никнейм и фракция пользователя 
    - рядом с жизнями указано общее количество карт в руке 
  - Показана карта лидера и есть возможность ее просмотреть и использовать 
  - Поле с отбоем (кладбище карт) отбой можно просмотреть нажав на него 
    - содержит вышедшие из игры карты 
    - последняя карта отображается 
  - Поле с колодой карт 
    - отображается количество оставшихся карт 
    - отображается рубашка карты в зависимости от фракции игрока 
  - Поле с картами в руке 
    - можно взаимодействовать с любой из карт (также для хода) 
    - при выборе карты появляется описание способности если оно есть 
  - Реализована сборка карты 
    - каждая карта собирается в соответствии со своим типом (карта с подробным описанием, карта боевая на поле, карта для сборки колоды с количеством ее экземпляров) 
    - карта лидеров 
    - карта героев 
    - карта принадлежащая фракции отображается с соответствующим баннером фракции 
    - отображается тип поля карты, ее способности и урон (если он есть) 
  - Доска с игрой 
    - на поле усиления можно ставить только “Командирский рог”, она удваивает урон на поле напротив 
    - на поле определенного типа можно ставить только карту такого же типа, на заднем фоне поля отображается символ обозначающий его тип
    - реализовано поле для карт погоды 
    - во время вашего хода можно просматривать карты на поле и вражеского лидера (дабл клик по линии) +
  - Присутствует возможность сдаться, после нажатия показывается статистика о текущем матче и оповещение о победе/проигрыше (кнопку "сдаться" нужно удерживать )
  - Есть возможность спасовать 
    - после того как игрок спасовал ход передается другому игроку, этот игрок может сделать неограниченное количество ходов. Когда второй игрок нажмет тоже “спасовать” происходит пересчет очков и игроку с наименьшим количеством очков отнимается жизнь(один из рубинов становится серым)
  - Можно включить музыку 

## Реализован мультиплеер 
  - В начале хода определяется кто будет ходить первым
    - если оба скоя'таэли то выбирается случайным образом
    - если один из двух игроков относится к фракции скоя'таэли то он выбирает за кем первый ход в начале матча
    - если оба игрока не относятся к фракции скоя'таэли то за кем ход определяется случайным образом
  - После определения кто ходит оба игрока могут заменить две карты из своей руки на представленные в карусели 
  - Каждый ход сопровождается соответствующими баннерами 
    - пока ходит один второй не может сделать свой ход, но может просматривать карты
  - Визуально выделяется ход одного из игроков подсветкой рамки с информацией об игроке так же всплывает баннер с информацией о том кто ходит 
  - реализованы способности фракций и синхронизированы с двумя пользователями, оповещение о срабатывании способности сопровождается - информационным баннером 
    - Скоя'таэли выбирают кто будет ходить первый
    - после каждого выигранного раунда фракция Севера берет карту в руку 
    - после каждого раунда фракция Монстров сохраняет случайную карту на столе 
    - после окончания раундов при случае ничьей фракция Нильфгаард одерживает победу
   
## Способности карт и лидеров 
  - Реализованы и синхронизированы способности карт 
    1) Проворство - Можно выложить в рукопашном или дальнобойном ряду. После этого передвигать нельзя.
    2) Прочная связь - Выложите рядом с картой с тем же названием, чтобы удвоить силу обеих карт.
    3) Чучело - Обменяйте на уже выложенную карту, чтобы вернуть ее в руку. 
    4) Медик - Выберите одну карту из вашего отбоя (не героя и не специальную карту и сыграйте ее немедленно. 
    5) Шпион - Выложите на поле вашего противника (сила карты добавится к очкам вашего противника) и возьмите две карты из вашей колоды. 
    6) Командирский рог - Удваивает силу всех отрядов в своем ряду. В ряду может быть только 1 такая карта.
    7) Мастер - Находит в своей колоде карты с таким же названием и мгновенно разыгрывает их. 
    8) Казнь - Уничтожает самый сильный отряд напротив (или отряды) вашего врага, если общая сила этих отрядов равна или превышает 10. 
    9) Ливень - Снижает силу всех осадных отрядов до 1 у обоих игроков. 
    10) Мгла - Снижает силу всех дальнобойных отрядов до 1 у обоих игроков. 
    11) Ясное небо - Отменяет эффекты всех погодных карт и очищает поле специальных карт (Мороза, Мглы и Ливня). 
    12) Мороз - Снижает силу всех рукопашных отрядов до 1 у обоих игроков. 
    13) Специальная казнь - Уничтожает самые сильные карты (или карту) на поле боя. Сыграв, отправьте в отбой. 
    14) Прилив сил - Дает +1 всем отрядам в ряду *кроме себя самого).

  - Реализованы и синхронизированы способности всех лидеров 
    1)  Фольтест Король Темерии: Возьмите из вашей колоды Мглу и сыграйте эту карту немедленно., 
    2) Фольтест Предводитель Севера: Отменяет все эффекты погодных карт(Мороза Ливня или Мглы)., 
    3) Фольтест Завоеватель: Удваивает силу всех ваших осадных отрядов(если в этом ряду нет Командирского рога)., 
    4) Фольтест Железный Владыка: Уничтожает самый сильный осадный отряд(или отряды) противника если общая сила его осадных отрядов равна или превышает 10., 
    5) Эмгыр вар Эмрейс Йож из Эрленвальда: Возьмите из вашей колоды Ливень и сыграйте эту карту немедленно.,
    6) Эмгыр вар Эмрейс Император Нильфграада: Посмотрите три случайные карты из руки противника., 
    7) Эмгыр вар Эмрейс Белое Пламя: Отменяет эффекты умения вражеского лидера.,
    8) Эмгыр вар Эмрейс Властелин Юга: Возьмите карту из отбоя вашего соперника., 
    9) Францеска Финдабаир Истинная эльфка: Возьмите из вашей колоды Мороз и сыграйте эту карту немедленно., 
    10) Францеска Финдабаир Маргаритка из Долин: Возьмите дополнительную карту в начале боя., 
    11) Францеска Финдабаир Прекраснейшая: Удваивает силу всех ваших дальнобойных отрядов(если в этом ряду нет Командирского рога)., 
    12) Францеска Финдабаир Королева Дол Блатанны: Уничтожает самый сильный рукопашный отряд(или отряды) противника если общая сила его рукопашных отрядов равна или превышает 10., 
    13) Эредин Бреакк Глас Король Aen Elle: Возьмите из вашей колоды любую погодную карту и сыграйте ее немедленно., 
    14)  Эредин Бреакк Глас Командир Дикой Охоты: Удваивает силу всех ваших рукопашных отрядов(если в этом ряду нет Командирского рога)., 
    15) Эредин Бреакк Глас Владыка Тир на Лиа: Сбросьте две карты и возьмите из вашей колоды одну карту по выбору., 
    16)  Эредин Бреакк Глас Убийца Оберона: Верните карту из вашего отбоя в руку.,
